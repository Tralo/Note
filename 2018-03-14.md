##  RxJava学习记录


### RxJava是什么
----------------------
> RxJava 英文介绍是"a library for composing as asynchronous and event-based programs using obserable sequences for the Java VM"(一个在Java VM上使用可观测的序列来组成异步的，基于事件的程序的库)。这就是RxJava，本质是异步，是一个实现异步操作的库。


### RxJava使用的好处
> 简洁性，Android系统中，AsyncTask和Handler都是为了让异步代码更加简洁。RxJava的优势也是简洁，但是即使随着程序逻辑变得越来越复杂，它依然能够保持简洁。
