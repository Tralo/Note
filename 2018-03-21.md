# Kotlin语法记录
### 1)?的含义: 在kotlin中单独使用? 表示可为空
> kotlin代码书写格式:
```Kotlin
var result = str?.length

```
> java代码书写格式:
```Java
if(str == null)
	result = null;
else
	result = str.length;

```

### 2)!!的含义：在kotlin中表示一定不能为空
```kotlin
var str:String? = null
var result : Int = str!!.length
```

### 3)?:的含义:在kotlin中表示三元操作符(即三目运算符)
```kotlin
var str:String?=null
var result = str?.length?:-1
//等价于
var result:Int = if(str != null) str.length else -1


```
